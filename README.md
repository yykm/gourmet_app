## Gourmetについて

ぐるな〇のようなグルメ情報サイトを模倣した転職活動用ポートフォリオ。

* フリーワード・現在地より範囲を指定して飲食店
* 写真投稿
* 口コミ投稿
* お気に入り
* 予約（開発環境のみ予約確認メール送信機能）
* Google Mapより店舗所在地の確認

などの機能を疑似的に実装しました。
